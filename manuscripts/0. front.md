Convention
這是代碼：

``` python
'''This is a sample python code.'''
def sum(a, b):
    return a+b
print( sum(1,2) )
print( sum(10,30) )
```

這是於 Command Line Prompt 運行的指令，開頭有個 $ 符號代表是 Command Prompt，但不用輸入的：

```
$ python example.py
```

檔案名稱或程序用名字於內文使用斜體顯示，例如：example.py 內的 today 函數。
